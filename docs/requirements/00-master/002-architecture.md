## 🏗️ 架构设计

### 核心架构
```
┌─────────────────────────────────────────────────────────────┐
│                      应用层 (Application)                    │
├─────────────────────────────────────────────────────────────┤
│  版图编辑器 (主程序)           插件管理器                    │
│  ┌─────────────────────┐      ┌─────────────────────┐      │
│  │  原理图插件 (可选)   │      │  其他插件 (可选)     │      │
│  └─────────────────────┘      └─────────────────────┘      │
├─────────────────────────────────────────────────────────────┤
│                      核心层 (Core)                           │
│  版图数据库 │ 几何引擎 │ DRC 引擎 │ 数据导入/导出            │
├─────────────────────────────────────────────────────────────┤
│                      基础层 (Foundation)                     │
│  Qt5 GUI │ C++ STL │ 文件系统 │ 网络库                       │
└─────────────────────────────────────────────────────────────┘
```

## 📐 技术选型

| 模块 | 技术 | 理由 |
|------|------|------|
| 语言 | C++17/20 | 高性能、内存控制 |
| GUI 框架 | Qt5.15+ | 跨平台、成熟稳定 |
| 构建系统 | CMake | 跨平台构建 |
| 测试框架 | GoogleTest | 单元测试 |
| 插件系统 | Qt Plugin | 动态加载 |

## 🔌 插件架构

### 插件接口
```cpp
class IPlugin {
public:
    virtual QString name() const = 0;
    virtual QString version() const = 0;
    virtual void initialize() = 0;
    virtual void shutdown() = 0;
    virtual QWidget* mainWidget() { return nullptr; }
};
```

### 插件加载
- 启动时扫描 `plugins/` 目录
- 动态加载 `.dll`/`.so` 文件
- 注册到插件管理器

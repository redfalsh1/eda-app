**版本**: v1.0

## 📋 描述

EDA 应用的主窗口，采用 QMainWindow 框架，包含菜单栏、工具栏、状态栏和中央区域。

---

## 🖼️ UI 布局

```
┌─────────────────────────────────────────────────────────────┐
│  菜单栏 (QMenuBar)                                           │
│  [文件] [编辑] [视图] [工具] [插件] [帮助]                    │
├─────────────────────────────────────────────────────────────┤
│  工具栏 (QToolBar)                                           │
│  [新建] [打开] [保存] | [撤销] [重做] | [缩放] [平移]         │
├──────────┬────────────────────────────────────┬─────────────┤
│          │                                    │             │
│  项目    │         版图绘制编辑器              │   图层      │
│  文件树  │         (QGraphicsView)            │   管理      │
│  (左侧)  │                                    │   (右侧)    │
│  (Dock)  │                                    │   (Dock)    │
│          │                                    │             │
├──────────┴────────────────────────────────────┴─────────────┤
│  状态栏 (QStatusBar)                                         │
│  [坐标：X,Y] [缩放：100%] [图层：1] [选中：0]                │
└─────────────────────────────────────────────────────────────┘
```

---

## 📐 功能要求

### 菜单栏
- **文件**: 新建/打开/保存/导出/退出
- **编辑**: 撤销/重做/剪切/复制/粘贴
- **视图**: 缩放/平移/网格/标尺
- **工具**: DRC/测量/属性编辑
- **插件**: 插件管理/加载/卸载
- **帮助**: 文档/关于

### 工具栏
- 常用操作快捷按钮
- 支持自定义工具栏
- 支持工具栏停靠

### 状态栏
- 实时显示光标坐标
- 显示当前缩放比例
- 显示当前图层
- 显示选中对象数量

---

## 🔧 技术要求

- 使用 `QMainWindow` 作为基类
- 支持布局状态保存/恢复
- 支持高 DPI 适配
- 支持深色主题

## 📐 功能要求

### 图层列表
- 显示图层编号
- 显示图层名称
- 显示可见性 (复选框)
- 显示颜色预览

### 图层控制
- 切换可见性
- 修改颜色
- 设置当前图层
- 图层过滤

### 批量操作
- 全选/反选/清空
- 批量可见性设置
- 图层排序

### 图层属性
- 新建图层
- 删除图层
- 重命名
- 属性编辑

---

## 🔧 技术要求

- 使用 `QListView` 或 `QTreeWidget`
- 自定义委托 (Delegate) 渲染颜色
- 使用 `QColorDialog` 选择颜色
- 支持图层过滤 (QSortFilterProxyModel)

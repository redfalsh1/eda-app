# REQ-203 - 项目文件树

**版本**: v1.0  
**优先级**: 🔴 高  
**状态**: 🔄 开发中  
**最后更新**: 2026-03-30

---

## 📋 描述

左侧项目文件树面板，显示项目层次结构和 Cell 列表。

---

## 🖼️ UI 示意

```
┌─────────────────────────────┐
│ 📁 项目文件树                │
├─────────────────────────────┤
│ 📁 eda-project              │
│ ├─ 📄 top_cell             │
│ ├─ 📄 cell_analog          │
│ ├─ 📄 cell_digital         │
│ │   ├─ 📄 sub_cell1       │
│ │   └─ 📄 sub_cell2       │
│ └─ 📁 libraries            │
│    ├─ 📁 tech_lib          │
│    └─ 📁 user_lib          │
└─────────────────────────────┘

右键菜单:
├─ 新建 Cell
├─ 重命名
├─ 删除
├─ 导入
├─ 导出
└─ 属性
```

---

## 📐 功能要求

### 树形结构
- 项目根节点
- Cell 节点 (可嵌套)
- 库节点
- 文件节点

### 操作支持
- 展开/折叠
- 单选/多选
- 拖拽排序
- 右键菜单
- 双击打开

### 状态显示
- 修改状态 (*)
- 锁定状态 (🔒)
- 引用计数

---

## 🔧 技术要求

- 使用 `QTreeView` + `QStandardItemModel`
- 支持自定义委托 (Delegate)
- 支持拖拽 (Drag & Drop)
- 支持上下文菜单

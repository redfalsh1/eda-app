## 📋 描述

使用 QDockWidget 实现可停靠、可浮动、可拖拽的面板系统。

## 📐 功能要求

### 项目文件树面板 (左侧)
- 显示项目层次结构
- 支持展开/折叠
- 支持右键菜单
- 支持拖拽排序
- 支持双击打开

### 图层管理面板 (右侧)
- 显示所有图层列表
- 图层可见性切换
- 图层颜色设置
- 当前图层选择
- 支持图层过滤

### 通用功能
- 拖拽浮动
- 重新停靠
- 自动隐藏
- 关闭/打开
- 状态保存/恢复

---

## 🔧 技术要求

- 使用 `QDockWidget` 基类
- 设置 `DockWidgetMovable` | `DockWidgetFloatable` | `DockWidgetClosable`
- 使用 `QMainWindow::saveState()` / `restoreState()`
- 支持多个面板同时浮动

## 📐 功能要求

### 画布功能
- 无限画布 (虚拟坐标)
- 网格显示 (可配置间距)
- 标尺显示 (上下左右)
- 对齐辅助线
- 吸附功能 (Grid/Object)

### 图形绘制
- 矩形 (Rectangle)
- 多边形 (Polygon)
- 路径 (Path)
- 文本 (Text)
- 实例引用 (Instance)

### 编辑操作
- 选择 (单选/框选/多选)
- 移动 (拖拽/键盘)
- 复制/粘贴
- 删除
- 撤销/重做
- 旋转/镜像

### 视图操作
- 缩放 (鼠标滚轮/快捷键)
- 平移 (中键拖拽)
- 适应窗口
- 跳转到对象

---

## 🔧 技术要求

- 使用 `QGraphicsView` + `QGraphicsScene`
- 纳米级精度 (坐标使用 int64_t)
- 支持大规模图形 (100 万+)
- 高效渲染 (LOD 技术)
- 支持 OpenGL 加速
